# civic-guide-india
🗳️ CivicPulse: Interactive Election Guide
A modern, AI-powered concierge for the 2026 election cycle.
Built with Next.js, Tailwind CSS, and Claude 3.5 Sonnet, CivicPulse transforms complex voting bureaucracy into a high-end, "Deep Forest" themed interactive experience.
✨ Features
Progress Tracking: Visual representation of your journey toward being "Election Ready."
Nature-Tech UI: A custom "Deep Forest & Gold" aesthetic using glassmorphism and modern typography.
Real-time Intelligence: Powered by Claude 3.5 to provide hyper-local election steps, deadlines, and requirements.
Action Chips: One-tap quick actions for registration and polling information.
🛠️ Tech Stack
Framework: Next.js 14+ (App Router)
Styling: Tailwind CSS
AI Engine: Anthropic Claude 3.5 Sonnet
Deployment: Optimized for Vercel, Netlify, or Cloudflare Pages
🚀 Quick Start
1. Clone the repository
 git clone https://github.com/YOUR_USERNAME/civic-pulse-app.git
cd civic-pulse-app
npm install
ANTHROPIC_API_KEY=your_sk_ant_key_here
npm run dev
Open http://localhost:3000 to see the app.
☁️ Deployment Instructions
Deploy to Netlify / Vercel
Push your code to a GitHub repository.
Connect your repository to your chosen cloud platform.
Critical: Add the ANTHROPIC_API_KEY under the "Environment Variables" section in your dashboard settings.
Deploy. The platform will automatically detect the Next.js build settings.
